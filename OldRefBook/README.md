# Old Reference Book Source Material

This folder contains the original reference book photos/content that will be processed and converted to modern topic-based modules.

## Folder Structure

- Photos are organized by chapter/topic in subfolders (01_Chapter_Photos through 09_Chapter_Photos)
- `patient-assessment/` - Patient assessment reference pages

## Processing

Photos will be analyzed using AI (Grok) to extract and verify text content, which will then be:
1. Converted to modern topic-based modules
2. Rebranded to Black Elk Mountain Medicine
3. Updated from SOAA'P to SOAPNote format
4. Randomized with US National/State Park locations for scenarios
5. Organized into 9 main modules maintaining the original chapter structure
